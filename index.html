FAQs

How can I create my own Poster server?

Creating a server is simple, but often requires wait time because you submit a Issue on our GitHub page.

1. Create a github repository

2. Create your own web page

3. Deploy your web page in GitHub Pages (thats required or else we will reject your request)

4. Submit a Issue on our GitHub page

5. Wait for us to review your request

6. Once approved, your data will be available in Poster.

Your code must be open source, and have a link to code, or else we will also reject your request.
